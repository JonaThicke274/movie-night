# movie-night
Cinematic Spirits

## Purpose
An application that matches a user's chosen film or a randomly generated film to a thematically appropriate cocktail by matching keywords from the film's plot to drink ingredients.


## Built With
* HTML
* CSS
* JavaScript
* Bulma
